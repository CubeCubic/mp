```markdown
# mp — Персональный музыкальный сайт (для CubeCubic)

Этот репозиторий содержит backend (Node/Express) и frontend (HTML/CSS/JS) для персонального сайта с возможностью:
- добавлять/редактировать/удалять треки, обложки, тексты (через веб-админку);
- группировать треки в альбомы;
- слушать треки в браузере (streaming с поддержкой range);
- скачивать треки через прямую ссылку;
- ставить лайки;
- проигрыватель появляется внизу страницы только при воспроизведении.

Вся инструкция ниже — шаг за шагом, как развернуть локально и на render.com бесплатно.

ВАЖНО: В админке есть пароль (ADMIN_PASS). Настройте переменные окружения (на Render и локально).

Структура (предполагаемая):
mp/
├── backend/
│    ├── app.js
│    ├── package.json
│    ├── tracks.json
│    ├── uploads/   ← сюда загружать mp3/jpg и т.д.
│    └── .gitignore
├── frontend/
│    ├── index.html
│    ├── admin.html
│    ├── app.js
│    └── styles.css
└── README.md

Дальше детально: как запустить локально, как развернуть на render.com, как добавлять треки, где менять текст UI и т.д.
```
